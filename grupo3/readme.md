
## Instalar

Usamos virtualenv para crear el ambiente de python y pip para instalar los requerimientos
```bash
$ pip install virtualenv
$ virtualenv venv
$ . venv/bin/activate
$ cd app
$ pip install -r requirements.txt
$ python app.py
```

## Ejecutar

```bash
$ python app/app.py
```

## Cerrar venv
```bash
$ deactivate
```
